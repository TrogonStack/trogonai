# AgentImplementation

The pinned code that interprets an agent definition (for example Codex or Claude
Code), selected as an exact `AgentImplementationVersion`. Hosting an
implementation is not itself an implementation. See
[ADR#0031](../adr/0031-agent-implementation-and-session-plan.md).
