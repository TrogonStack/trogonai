# ARD

Agent Resource Discovery. An HTTP and JSON specification for cataloging agentic
resources such as A2A agent cards, MCP server cards, catalogs, and registries.
Used here as an external, discovery-only compatibility boundary. See
[ADR#0012](../adr/0012-ard-compatible-discovery-catalog.md).
