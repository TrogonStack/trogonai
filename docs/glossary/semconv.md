# semconv

Semantic conventions. The registry of named attributes, spans, and metrics that
telemetry must use so signals stay consistent across the platform. See
[ADR#0008](../adr/0008-opentelemetry-observability.md) and the Observability
section of [Decider Platform](../architecture/decider.md#observability).
