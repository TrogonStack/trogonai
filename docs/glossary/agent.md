# Agent

The stable identity of an autonomous agent. It carries no implementation family
or model; those live in configuration so that changing them is a reviewed change.
See [ADR#0025](../adr/0025-agent-definition-data-ownership.md) and
[ADR#0031](../adr/0031-agent-implementation-and-session-plan.md).
