# AAuth

The agent authentication scheme (`draft-hardt-oauth-aauth-protocol`) giving every
agent its own cryptographic identity bound to a signing key, verifiable without
pre-registration. Bound to a Trogon NATS proof-of-possession binding. See
[ADR#0017](../adr/0017-aauth-agent-authentication.md).
