# Transport

A binding that frames and delivers protocol messages. Examples: stdio,
Streamable HTTP, WebSocket, a NATS bridge. See
[ADR#0003](../adr/0003-ai-protocol-transport-taxonomy.md) and
[ADR#0004](../adr/0004-protocol-and-transport-layering.md).
