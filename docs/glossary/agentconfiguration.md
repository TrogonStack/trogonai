# AgentConfiguration

The typed record holding an agent's exact implementation selection, primary and
auxiliary model selections, and the rest of its behavior. Digest-committed, so
any behavior-significant change produces a new revision. See
[ADR#0031](../adr/0031-agent-implementation-and-session-plan.md).
