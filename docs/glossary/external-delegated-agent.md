# External delegated agent

A remote system the platform cannot attest as running the planned implementation
and model, so it is treated as an opaque delegation target with a recorded result
rather than a verified session. See
[ADR#0031](../adr/0031-agent-implementation-and-session-plan.md) and
[ADR#0032](../adr/0032-model-route-and-credential-binding.md).
