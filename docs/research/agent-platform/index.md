# Agent platform research corpus

This corpus is the frozen research input behind the platform's
agent-architecture decisions: an industry study of agent products plus the
running record of the design discussion that followed it. Where a
conclusion here differs from an accepted record in the
[ADR index](../../adr/index.md), the ADR is authoritative.

## Method

The [research prompt](./RESEARCH_PROMPT.md) is preserved so the shared scope
and evidence rules behind each product dossier remain reproducible.
